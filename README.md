# KaalisBank

An academic front-end project (HTML/CSS/JavaScript) simulating a digital banking website for the UEMOA region, built as an end-of-year assignment. I chose a banking theme out of interest in the fintech/banking IT sector.

It includes a landing page with a working XOF/EUR/USD/GBP converter, a simulated login, and a dashboard with mock account data and a second converter.

Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no backend. Typography uses IBM Plex Serif/Sans/Mono, and the design (navy + gold, sharp corners, two-tier nav) is inspired by real regional bank sites rather than generic templates.

This is a demo project only — not a real bank.
